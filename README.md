# Mohit Kashyap – MERN Developer Portfolio

A modern animated portfolio website built with React + Vite + TailwindCSS + Framer Motion.

## Run Locally in VS Code

1. Install **Node.js** (v18 or newer): https://nodejs.org
2. Open this folder in **VS Code**
3. Open the integrated **Terminal** (Ctrl + `)
4. Run these commands:

```bash
npm install
npm run dev
```

5. Open your browser at **http://localhost:5173**

## Build for Production

```bash
npm run build
npm run preview
```

The compiled site appears in the `dist/` folder — ready to host anywhere (Vercel, Netlify, GitHub Pages, etc.).

## Tech Stack

- React 18 + TypeScript
- Vite 6 (lightning-fast build tool)
- Tailwind CSS v4
- Framer Motion (animations)
- Radix UI + shadcn/ui components
- Lucide React (icons)
